localStorage Explained

What is localStorage?

localStorage allows developers to store web app data in the user's browser for an infinite ammount of time.

This means the browser can be closed and opened again anytime in the future and the data will remain stored.

The Pros

  • Data does not expire
  • Can store 5-10MB of data
  • Simple to use
  • Eliminates need for a backend database

The Cons

  • Stores a maxiumum of 10MB
  • Data is not secure
  • Limited to storing strings
